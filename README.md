GeoFS-Radar

Simulates radar for ingame flight.

'q': start radar
'w': select target forward
'r': select target backward
'l': lock on target
'y': increase radar range
'u': reduce radar range

When radar is activated, all player name tags are removed. 
Locking on targets will show the name tag and their information, in the format of 'name' 'speed' 'closing speed(hot+,cold-)' 'altitude'.




Attribute to GeoFS Blue angels F18 mod and multiplayer info mod.
